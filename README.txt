This application uses OAuth to authenticate users. This OAuth flow uses passport to interact with the Google server on behalf of the user simplifying your integration with Google APIs.


Any application that uses OAuth 2.0 to access Google APIs must have authorization credentials that identify the application to Google's OAuth 2.0 server.
